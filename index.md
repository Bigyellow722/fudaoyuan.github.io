#  解题思路

- 考试中遇到情景题的解题思路：
	1. 调查研究事情的真相，确定事情的严重程度，具体问题具体分析
	2. 考虑是否要向上级反映，汇报

- 考试中遇到学生相关问题的解题思路：
	1. 要以学生为主体考虑问题，思考如何稳定学生情绪，有效与学生交流沟通
	2. 问题是否具有普遍性，能否利用这个机会加强对其他学生的教育和指导
	3. 是否需要与家长沟通
	4. 如果问题比较严重，是否需要请求心理咨询机构专业人员（公安人员，或保卫处，或学校学生会）等外力支持和援助
    5. 事后如何总结，提高对学生事件(突发事件)的处理能力
